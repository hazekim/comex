Adaptive Background System: Optimized backgrounds for different screen ratios.

Responsive Navigation: A fixed-top Bootstrap navbar for easy access to sections. 
Interactive Contact Form: A clean, shadow-styled interface for client queries. 
Modern Typography: Utilizing Google Fonts (Montserrat) for a professional look. 

üì± Mobile Optimization 
The project specifically addresses the challenge of 9:16 aspect ratio displays. 
Backgrounds are locked to 100vh to prevent overflow issues. 
Uses object-fit: cover and object-position: center top to maintain image quality without distortion when the window is resized. 

üîí Security & Privacy (Important) 
Credential Protection: For security purposes, all sensitive SMTP authentication data (passwords, host IPs) in send_email.php has been removed or replaced with placeholders. 
Data Privacy: Any internal company server addresses or private contact information mentioned in the development phase have been obfuscated. 
Portfolio Usage: This repository is intended for portfolio demonstration. The core logic remains, but production-level secret keys are not included.
